REQUISITOS FUNCIONAIS 

RF01: Garantir que os dados estão completos e corretos 

RF02: Automatizar o momento de percorrer os dados 

RF03: Identificar alunos com baixo e alto rendimento acadêmico 

RF04: Gerar o relatório final 

RF05: Calcular média 

REQUISITOS NÃO FUNCIONAIS 

RNF01: Dividir função em cada branch de versionamento 

RNF02: Tratar erros na lista 

RNF03: O sistema deve ser modularizado

RNF04: O código deve ser organizado

REGRAS DE NEGÓCIO 

RN01: Alunos com média de noras abaixo de 7 possuem status de “Recuperação” 

RN02: Aluno com a maior média é classificado como "Top Student”. 

RN03: A média do aluno é calculada pela soma das notas dividida pela quantidade de notas

RN04: Cada aluno pode ter quantidade variável de nota

RN05: Dados inválidos não devem comprometer o processamento geral.
